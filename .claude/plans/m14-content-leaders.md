# Read-Thai — Unit 12 Content Bank: Silent Tone-Leader Words

Standard (Central) Thai. Two silent-leader phenomena covered:

1. **Silent HIGH-class ห** before a Low-class sonorant (ง ญ น ม ย ร ล ว). ห is not pronounced; the syllable follows **HIGH-class** tone rules.
2. **Silent MID-class อ** leading ย in exactly four everyday words. อ is not pronounced; the syllable follows **MID-class** tone rules. thai-language.com (id/212494, id/143076) explicitly confirms these are the only four: อยู่ อย่า อย่าง อยาก.

**Tone diacritic convention (this course / Tingsabadh & Abramson):** macron ◌̄ = mid, grave ◌̀ = low, circumflex ◌̂ = falling, acute ◌́ = high, caron ◌̌ = rising. Long vowel `ː`; unreleased stop finals `k̚ t̚ p̚`.

**Class tone-rule reference used in the derivations**
- HIGH class: live + no mark → **rising**; dead + no mark → **low**; + mai ek → **low**; + mai tho → **falling**.
- MID class: live + no mark → **mid**; dead + no mark → **low**; + mai ek → **low**; + mai tho → **falling**.
- "Live" = open long-vowel syllable or sonorant final (ง ญ น ม ย ว ...); "Dead" = stop final (ก ด บ ...) or short open vowel.

**Reading of thai-language.com transcription tone letters:** `M`=mid, `L`=low, `F`=falling, `H`=high, `R`=rising (used below as the secondary cross-check).

## Notes, disagreements, and drops

- **Helper-model tone mislabels (corrected, not a source disagreement).** The page-summariser repeatedly named contour `˩˩˦` "falling" and `˨˩` "falling." In the Tingsabadh & Abramson scheme used by Wiktionary, `˩˩˦` (114) is the **rising** tone and `˨˩` (21) is the **low** tone. This is confirmed internally by each Wiktionary entry's own Paiboon romanization (caron ◌̌ = rising for mǎa/nǎa/yǐng/…; grave ◌̀ = low for yàa/yàak/…) and externally by thai-language.com's tone letters (`R` = rising, `L` = low). All rows use the corrected tones.
- **ง (ngɔ) not represented.** There is no common, everyday silent-ห word on the ง sonorant (words like หงส์ hǒng "swan," หงาย ngǎai "face-up" are low-frequency/less transparent), so ง was intentionally left uncovered rather than seeding a rare word. Everyday ง-initial vocabulary (งู, งาน, ง่าย) uses ง as its own Low-class initial, not a ห-led one.
- **ให้ deliberately excluded.** ให้ /hâj/ "to give" is *not* a silent-leader word: the ห there is the actual pronounced HIGH-class initial /h/ (+ mai tho → falling), not a silent leader before a sonorant. Including it would teach the wrong mechanism.
- **หรือ — secondary source not independently captured.** thai-language.com's Thai-script search is not reachable through the available fetch/scrape path (the site presents a self-signed-cert error to WebFetch and its lexicon search defaulted to English mode via the scraper; a `หรือ` permalink did not surface in the site-scoped search that confirmed the other 11 words). It is retained on the strength of (a) the Wiktionary reading `/rɯː˩˩˦/` = rʉ̌ʉ (rising) and (b) an unambiguous orthographic derivation (ห-leader + live open long vowel + no tone mark → rising). No source disagreed; flagged only because the second source was not separately logged for this one row.
- **Source-tooling note.** Primary verification = en.wiktionary.org (WebFetch, all 12). Secondary = thai-language.com, obtained via site-scoped web search whose result snippets expose the site's own tone-lettered transcription (confirmed 11/12). thai2english.com was unavailable at verification time (site displayed a denial-of-service outage notice). No cross-source tone/spelling conflicts were found for any retained word.
